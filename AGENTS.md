# AGENTS.md

- Proyecto: Derbi Relampago, juego de futbol arcade multijugador online.
- Stack: HTML/CSS/JavaScript vanilla en `public/`, Node.js ESM con Express y `ws` en `server.js`.
- Instalar dependencias: `npm install` o en PowerShell si hay bloqueo de scripts, `npm.cmd install`.
- Ejecutar local: `npm start` o `npm.cmd start`; abre `http://localhost:3000`.
- Verificar sintaxis: `npm run check` o `npm.cmd run check`.
- Despliegue backend: Render Web Service usando `render.yaml`.
- Despliegue frontend: GitHub Pages publicando la carpeta `public/`.
